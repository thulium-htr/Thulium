# Language Support

Thulium is designed for global usage.

## Supported Languages

| Code | Language | Script | Features |
| :--- | :--- | :--- | :--- |
| `az` | Azerbaijani | Latin | Full support including ə, ğ, etc. |
| `tr` | Turkish | Latin | Dotted/dotless i handling. |
| `en` | English | Latin | Standard ASCII + punctuation. |
| `ru` | Russian | Cyrillic | Full Cyrillic alphabet. |
| `fr` | French | Latin | Accents (é, è, ê, etc.). |
| ... | ... | ... | ... |

*See `thulium/data/language_profiles.py` for the complete list.*
