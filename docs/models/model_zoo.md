# Model Zoo

Thulium supports a variety of architectures composed of different blocks.

## Backbones

- **ResNet**: Standard CNN backbone (ResNet18/34/50 variants).
- **ViT**: Vision Transformer for patch-based feature extraction.

## Sequence Heads

- **BiLSTM**: Bidirectional LSTM for standard sequence modeling.
- **Transformer**: Self-attention based sequence modeling.

## Decoders

- **CTC**: Connectionist Temporal Classification (Greedy, Beam Search).
- **Attention**: Autoregressive decoder for Seq2Seq tasks.
